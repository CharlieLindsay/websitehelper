# HelpMe Utilities Website
A simple website that helps streamline users workflow
A demonstration of the site running can be accessed at https://demo.charleilei.dpdns.org/

Everything is all over your device. There is no central place where utilties can be easily accessed, such as by pressing your browsers home button or switching tabs. That is why I have made a simple, easy to understand and comfortable HelpMe Utilties, which gives plenty of whitespace for the user to breathe. Some of the features are listed below. 

## Features
- Quick Site Access - Easily access commonly used sites!
- Edit button to change the quick site access
- Stores quick access for next time in local storage
- Time in analog format
- Weather based off browser location
- Stopwatch
- Localstorage-stored simple notepad tool
- Image insert spot for a commonly required image or timetable

## Requirements
- Localstorage is used in this website so must use up to date and common browser
- Javascript must be enabled
